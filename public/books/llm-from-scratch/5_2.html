Why Do We Need This?

The Shape Mismatch Problem

PyTorch's CrossEntropyLoss expects a 2D input of shape [N, C] where N is the number of samples and C is the number of classes. However, LLMs output 3D tensors: [Batch, Time, Vocab].

The loss function treats every token prediction as an independent event. It doesn't care about the sequence structureâ€”it just needs a list of guesses (logits) and a list of correct answers (targets).

Key Takeaway

Flattening converts a batch of sequences into a long list of individual token predictions. By merging the Batch and Time dimensions, we create a unified list where each row represents one token's logits, regardless of which sequence or position it came from. The order is preserved: Batch 0's tokens come first, then Batch 1's tokens are appended below.
